Domande frequenti

Che differenza c'è tra LoRa e WiFi per IoT?

LoRa è long-range (km) ma bassa banda (decine di kbps), ideale per sensori sparsi a basso traffico. WiFi è alta banda ma corto raggio. Per la mia tesi triennale ho lavorato su LoRa Mesh per IoT con pubblicazione IEEE CCNC 2023.

ESP32 o Raspberry Pi?

ESP32 per sensori autonomi a basso consumo (€5, batteria), Raspberry Pi per gateway/edge computing che richiedono OS Linux completo (€40+). Spesso si combinano: ESP32 sui nodi, Raspberry Pi come gateway.

Fate domotica custom?

Sì, basata su Home Assistant + MQTT + ESP32/ESPHome per i nodi. Integrabile con Apple HomeKit, Google Home, Alexa. Adatta a chi vuole privacy e controllo totale invece di cloud chiusi.

Sviluppate firmware in C/C++ o MicroPython?

Entrambi. MicroPython per prototipi rapidi e logica non performance-critical; C/C++ con ESP-IDF o Arduino framework per produzione, basso consumo o real-time.

Lavorate con il cloud per la telemetria IoT?

Sì, dashboard real-time con MQTT broker (Mosquitto, EMQX), backend Node.js o Go, storage time-series (InfluxDB, TimescaleDB), visualizzazione con Grafana.
