Domande frequenti

FastAPI, Django o Flask?

FastAPI per API performanti con type hints e doc OpenAPI automatica (mio default). Django per webapp full-stack con ORM e admin. Flask per microprogetti o quando vuoi controllo manuale.

Lavorate con AI e Machine Learning?

Sì. Sul progetto DISCOV.ER (Università di Bologna, PR FESR 2021-2027) ho lavorato su modelli di simulazione e previsione dati ambientali con AI/ML in Python. Esperienza anche su LLM e OCR AI-based.

Python è abbastanza veloce per produzione?

Per la maggior parte dei carichi sì. Per hot path performance-critical considero Go o Rust. Python + asyncio + uvloop + buon caching coprono il 95% dei casi.

Type hints in Python: obbligatori?

Per progetti che durano più di qualche mese: sì. Type hints + mypy o pyright migliorano sensibilmente refactoring e IDE. Performance: zero costo a runtime.

Avete esperienza con uv, ruff, pdm?

Sì. uv come package manager (10-100x più veloce di pip), ruff come linter+formatter unificato (sostituisce flake8, black, isort), pyright per type checking. Stack moderno end-of-2025/2026.
