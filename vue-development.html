Domande frequenti

Vue 2 o Vue 3?

Per nuovi progetti sempre Vue 3 con Composition API. Vue 2 è in EOL dal 2023 (con LTS pagata fino 2024). Migrazione da Vue 2 a Vue 3 è gestibile con il compat build.

Composition API o Options API?

Composition API è il default per nuovi progetti: meglio per riuso di logica (composables), TypeScript più solido. Options API resta valida per progetti semplici e curva di apprendimento più gentile.

Pinia o Vuex?

Pinia è il successore ufficiale di Vuex, raccomandato dal team Vue. API più semplice, TypeScript-first, devtools integrati. Per nuovi progetti sempre Pinia.

Nuxt 3 vs Vite + Vue?

Nuxt 3 se hai bisogno di SEO, SSR, file-based routing, auto-import. Vite + Vue se hai una SPA interna o vuoi controllo manuale. Per landing/marketing Nuxt + content modulo è imbattibile.

Quanto è grande la community Vue in Italia?

Vue è molto popolare in Italia per progetti freelance e PMI, meno presente in enterprise dove dominano React/Angular. Buona scelta per team che vogliono semplicità e produttività.
